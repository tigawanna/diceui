## Layout

Import the parts, and compose them together.

```tsx
import {
  TagsInput,
  TagsInputLabel,
  TagsInputItem,
  TagsInputItemText,
  TagsInputItemDelete,
  TagsInputInput,
  TagsInputClear,
} from "@diceui/tags-input";

return (
  <TagsInput>
    <TagsInputLabel/>
    <TagsInputItem >
      <TagsInputItemText />
      <TagsInputItemDelete />
    </TagsInputItem>
    <TagsInputInput />
    <TagsInputClear />
  </TagsInput>
)
```

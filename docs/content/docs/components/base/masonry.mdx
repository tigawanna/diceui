## Layout

Import the parts, and compose them together.

```tsx
import {
  Masonry,
  MasonryItem,
} from "@/components/ui/masonry";

return (
  <Masonry>
    <MasonryItem />
  </Masonry>
)
```

## Examples

### Linear Layout
